# Pattern Foundry Artifacts

This folder stores artifacts generated while using the `pattern-foundry` skill, such as UI audit screenshots, DOM extracts, accessibility snapshots, and implementation notes.

Do not analyze or load this folder during normal repo exploration. Only inspect it when the user invokes `pattern-foundry` or explicitly asks about Pattern Foundry audit/design work.

Current contents:

- `consensus-audit.md`: Pattern Foundry audit report and implementation handoff for `https://consensus.app/`
- `consensus-redesign-handoff.md`: original PAGE_GEN_MODE redesign handoff derived from the audit without cloning Consensus
- `consensus-desktop-full.png`: full-page desktop screenshot of `https://consensus.app/`
- `consensus-desktop-snapshot.md`: desktop accessibility snapshot
- `consensus-dom-audit.json`: extracted headings, CTAs, inputs, image alt text, and color usage
- `consensus-mobile-full.png`: mobile screenshot captured during the Consensus audit
- `consensus-mobile-snapshot.md`: mobile accessibility snapshot
- `consensus-mobile-390-full.png`: clean 390px mobile screenshot of `https://consensus.app/`
- `consensus-mobile-390-snapshot.md`: clean 390px mobile accessibility snapshot
- `consensus-sidebar-session-full.png`: responsive-session screenshot where the sidebar remained open after viewport changes
- `consensus-sidebar-session-snapshot.md`: accessibility snapshot for the sidebar-session state
